1. Keterkaitan SaaS, PaaS, dan IaaS dalam Cloud Computing

SaaS, PaaS, dan IaaS adalah tiga model layanan utama dalam cloud computing yang saling terhubung:

IaaS (Infrastructure as a Service)
Menyediakan infrastruktur dasar seperti VM, storage, dan jaringan.
Contoh: AWS EC2, Google Compute Engine.

PaaS (Platform as a Service)
Berjalan di atas IaaS. Menyediakan platform untuk menjalankan aplikasi (runtime, database, deployment).
Contoh: Heroku, Google App Engine.

SaaS (Software as a Service)
Berjalan di atas PaaS. Merupakan aplikasi siap pakai yang digunakan oleh end-user.
Contoh: Gmail, Google Docs, Canva.

Hubungan ketiganya:
IaaS menyediakan infrastruktur →
PaaS menyediakan platform di atas IaaS →
SaaS adalah aplikasi yang dibangun/diterbitkan melalui PaaS.
